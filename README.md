# EGS

Manycore EGS Graphics System main repo

## Requirements

- Node: >= 22.13.0
- typescript: ~3.8.3
- pnpm(corepack preferred)


## Repo Structure

- tools/ # script and utils used to build or generate code.
- packages/ # egs related typescript packages, see README.md under each package's directory for more information.
